1.游戏主要定位： <br/>

鉴于时间和成本限制，游戏主要定位在游戏性和创新性上。尽可能做好细节处理 <br/>
优化用户游戏体验，并通过难度的提高使玩家激发玩乐欲望并享受成就感。 <br/>

<br/>
2.游戏整体指导思路：跑酷+闯关 模式 <br/>

<br/>
传统的跑酷游戏游戏形式较为单一，动作设置上多为上（跳跃）、下（多为下滑 <br/>
或者快速降落，部分游戏没有向下的操作）、左、右（横版过关没有左右），有 <br/>
的只是简单的距离或分数上的堆叠。大部分游戏中会设置金币和道具，金币可以 <br/>
用来购买物品、形象或升级道具，道具用来提高分数。部分游戏会增添任务内容 <br/>
，完成任务会获得大量奖励。<br/>

<br/>
由此可以看出跑酷游戏的一些特点：即游戏性单一重复，不需要动脑子，操作简 <br/>
单，容易上手，缺少了趣味性。因而如果没有相关激励措施玩家会很容易感到疲 <br/>
倦厌烦，新鲜感一结束就删掉了。而这些元素（金币、道具、任务等等）都是设 <br/>
计者为了激励玩家进行再一回合的游戏，保持游戏的耐玩性而设置的。<br/>

<br/>
那么经典的闯关游戏如魂斗罗为何经久不衰呢，一方面是因为是童年的记忆，另 <br/>
一方面不同的地图，不同的闯关方式，BOSS的不同打法都告诉我们这不是简简单 <br/>
单凭着跳跃、开枪就能过关的游戏，还需要你花大量时间去研究如何去击杀BOSS <br/>
如何通关，而这一过程极大丰富游戏的趣味性的同时，还增加了玩家的游戏认同 <br/>
感，因而几年甚至十几年后仍然对它念念不忘。<br/>

<br/>
当然我们这款低成本的小游戏是难以做到这番成就的，但是这类闯关游戏的成功<br/>
却值得我们借鉴和运用。在魂斗罗中，玩家需要控制人物进行前进后退等操作，<br/>
而如果我们免去这些操作，只需要开枪和跳跃，这不就是一个横版跑酷游戏的雏<br/>
形嘛？事实上，闯关游戏中也有类似的游戏设置，在合金弹头、街机三国中都有<br/>
类似乘坐交通工具，不停前进，躲避障碍物的关卡。<br/>

<br/>
因为在这个思路下，产生了跑酷+闯关的游戏模式，游戏在原有的金币、道具、<br/>
任务等元素中添加BOSS，地图、关卡等元素，以期极大的丰富游戏性，具体设<br/>
计思路如下。<br/>

<br/>
3.BOSS如何设置？ <br/>

现有如下两种方案：<br/>
（1）BOSS的出现与距离相关，出现后会成为阻碍前进的障碍物之一，如果不<br/>
消灭掉会伴随直到下一个BOSS出现的判定距离为止。如设置每1000m出现一个<br/>
BOSS，BOSS的难度在不断增大。当跑到1000m时，会出现BOSSa，如果不消灭<br/>
掉，BOSSa会一直伴随，直到2000m时出现BOSSb为止，期间普通障碍物仍会按<br/>
照之前的频率出现（频率变化也可）。<br/>
（2）同样BOSS的出现与距离相关，不过当达到1000m时会进入一个异次元空<br/>
间，异次元空间不会计算距离，只有当消灭BOSS后才能回到普通跑酷模式。<br/>
<br/>
第一种方案容易走跑酷的空子，即当玩家反应迅速的话不需要击败BOSS，只<br/>
是在躲避障碍物的同时躲避BOSS即可，达到下一个判定距离自然会消失。当<br/>
然我们不可能设置成BOSS的攻击达到了已经无法前进的程度，这样虽然增加<br/>
了跑酷的难度，但缺少了研究如何击杀BOSS的乐趣。<br/>
<br/>
第二种方案恰好弥补了这个漏洞，但也同时产生了另一个问题，即玩到后期<br/>
BOSS难度很高的时候，如果难以短时间击杀BOSS，距离又不会增加会极大增<br/>
加玩家的挫败感，尤其当辛辛苦苦跑到后面，在BOSS的异次元空间中浪费大<br/>
量时间而实际距离又没有增加，如果因为BOSS而死亡，这种挫败感会更加强<br/>
烈。<br/>

<br/>
4.如何击杀BOSS？<br/>

没有BOSS，如何成为上忍？<br/>

如何击杀的前提即如何对BOSS造成伤害。<br/>
对BOSS造成的伤害的方式有以下几种方式：<br/>

（1）通过跳跃，直接跳到BOSS头上（参照马里奥）或跳到某种按钮触发机<br/>
关对BOSS造成伤害<br/>
（高难度设置1：可以设置不同的按钮，如触发红色机关对BOSS造成伤害，<br/>
触发绿色机关BOSS回血。）<br/>
（高难度设置2：可以设置不同的按钮，但是需要按照一定顺序踩到才会造<br/>
成伤害，不按顺序不会触发伤害）<br/>

（2）要尽可能获得某样物品对BOSS造成伤害，物品的获得方式可与道具的<br/>
获得方式类似。<br/>
（高难度设置1：物品的获得方式不是百分之百的，有的时候在获得物品的<br/>
同时会意味着撞上障碍物而死亡，因而玩家在是否去获得物品这一过程中<br/>
要有取舍。）<br/>
（高难度设置2：必须在一定时间内获得某一物品，未获得会意味着BOSS回<br/>
血或玩家死亡。）
<br/>
（3）通过武器进行伤害。<br/>
（高难度设置：需要不同武器转换进行相应攻击，比如BOSS在远处时需要<br/>
飞镖投掷，当BOSS跑到近处时需要刀剑劈砍进行击退。）<br/>
<br/>
当然击杀BOSS会获取大量金币，金币可以解锁人物、场景、武器等。<br/>

<br/>
5.武器系统。<br/>

忍者除了一身的忍术，还需要趁手的装备。<br/>

金币是可以购买武器的，每一轮游戏开始前要对武器进行选择，可以装备<br/>
的武器分为远程武器和近程武器，每一类型只能装备一种武器，如何区分<br/>
武器优劣？<br/>

武器可以通过攻击范围、伤害大小、攻击速度和频率来区分优劣。<br/>

（1）攻击范围：飞镖只能打到手机屏幕的一半，而手里剑可以打到手机<br/>
屏幕的最右侧。（当然飞镖这种远程武器不能因小攻击范围而无法消灭B<br/>
OSS，当然为了体现武器的优劣可以设置不断移动的BOSS，手里剑可以一<br/>
直对BOSS造成伤害，而飞镖只能当BOSS靠近时才能造成伤害。）<br/>

（2）伤害大小：武器要尽量做到平衡，高伤害意味着攻击范围小，攻击<br/>
速度慢。
<br/>
（3）攻击频率<br/>

当然武器的选择要与BOSS有关联。<br/>

例如选择刀剑攻速快，伤害低，利于击杀1000m的BOSSa,而选择斧头攻速<br/>
慢，伤害高，利于击杀2000m的BOSSb。选择斧头更有利于闯后面的关卡，<br/>
但是闯前面的关卡有花费一点时间，而且至少还要保证自己能跑到2000m。<br/>

<br/>
6.道具系统<br/>

忍者的世界充满危险，机遇与挑战并存。<br/>

传统的跑酷的游戏中的道具都是利于玩家得分的，但是分析一下为什么<br/>
麻将或赌博游戏如此着迷？其中这种游戏的随机性也就是所谓的“靠运<br/>
气”促使了游戏者不停地玩下去。因而将这种随机性引入我们的道具系<br/>
统中来，即获得的道具不全是有利的，会存在获得负面道具的可能性。<br/>

正面效应的道具比较常见，如保护罩、无敌加速一段时间、大量金币、<br/>
磁铁（设置磁铁感觉就不是忍者了）等。<br/>

负面效应的道具如下：<br/>
（1）跳的更高。跳的更高意味着无法跳跃两次，可能在一次跳跃降落<br/>
的时候恰好撞上障碍物而束手莫测。<br/>
（2）步伐变缓。挥动武器或跳跃都会比平时要慢一点，意味着要比平<br/>
时更早的发出指令。<br/>
（3）晕头转向。即手指下划的时候忍者会跳跃，手指上划的时候忍者<br/>
会下铲。<br/>

当然道具系统出现正面效应的道具的几率要大，这样才会激发积极性。<br/>

其他方案：<br/>
可以设置道具在箱子里，通过远程武器可以提前打碎箱子，正面效应<br/>
或负面效应的道具的样子会有所不同，玩家可以自行抉择是否去获取<br/>
该道具。<br/>

<br/>
7.难度设置。<br/>
难度设置可以从两方面来体现：游戏速度和障碍物。<br/>
（这两方面可以同时调节游戏速度，也可以只选择一方面。比如可以<br/>
设置游戏的速度是固定不变的，初始速度就是比较高的，但是由于后<br/>
期障碍物的增多以及BOSS来增加难度。）<br/>
<br/>
（1）游戏速度：跑酷游戏的游戏速度都是递增的，游戏一开始不能<br/>
太快（太快没有上升空间），同样不能太慢（太慢会影响游戏体验）<br/>
在这方面要进行后期测试，以期设置较为适宜的游戏初始速度以及<br/>
速度的变化率。<br/>
<br/>
（2）障碍物：可以通过障碍物的数量或增加躲避难度来增加难度。<br/>
增加躲避难度即玩家需要一定难度的组合操作才能躲避，例如会突<br/>
然跳出来一个敌方忍者扔下三个飞镖，要快速的下铲、跳起再下铲<br/>
才能躲避等（高难度设置：跳起的敌方忍者会随机投掷飞镖）。<br/>

<br/>
8.细节优化。<br/>
主题是忍者，要尽量还原忍者风格。<br/>
如忍者跳跃的时候需要有忍术，瞬移加速的时候要有幻影等。<br/>
